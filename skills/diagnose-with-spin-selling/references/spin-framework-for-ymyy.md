# SPIN Framework For 有膜有漾

This reference adapts Neil Rackham's SPIN Selling framework to C-end car-film sales.

## Core Adaptation

The original SPIN framework was designed for larger, consultative sales. In car-film retail, the conversation is usually shorter, so use a compressed version:

1. One or two Situation questions.
2. One or two Problem questions.
3. One Implication question only when the customer has a real pain.
4. One Need-payoff question before recommending.

## Need Types

- Implied need: the customer expresses a problem, dissatisfaction, concern, or risk.
  - Example: "夏天车里太热了。"
  - Sales response: diagnose further; do not immediately pitch.
- Explicit need: the customer expresses a desired outcome.
  - Example: "我想要隔热好一点，后排也有隐私。"
  - Sales response: recommend a matching product and explain value.

## Capability Language

Avoid leading with features. Translate features into customer outcomes.

| Feature-heavy | Customer-value version |
|---|---|
| 10mil 厚度 | 更适合重视长期漆面保护的客户 |
| 紫外线阻隔率 | 减少暴晒和后排乘坐不适 |
| 电脑裁膜 | 降低现场动刀和拆卸带来的风险 |
| 质保期 | 后续使用更安心，具体按产品来源页确认 |

## Commitment Ladder

Do not force a close too early. Choose the smallest useful next step:

- Send model and budget.
- Compare two packages.
- View sample film.
- Check same-model case.
- Visit store for inspection.
- Confirm construction plan.

## Anti-patterns

- Starting with price before diagnosis.
- Asking too many Situation questions.
- Treating "我想了解一下" as a buying signal.
- Answering a Problem signal with a product dump.
- Using exaggerated fear to push paint protection film.
- Calling a feature a benefit without connecting it to the customer's stated need.
