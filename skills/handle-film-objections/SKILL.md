# Handle Film Objections

## Overview

Use this skill to help sales respond to customer concerns about 有膜有漾 products and services in a calm, factual, commercially useful way.

## Triggers

Use this skill when the customer says or implies:

- 太贵了
- 有没有必要贴
- 会不会伤车
- 质保多久
- 贴完能不能洗车/跑高速
- 别家更便宜
- 贴了会不会影响雷达

## Workflow

1. Restate the concern in one sentence.
2. Separate facts from policy.
   - Product specs and aftercare: answer from knowledge base and cite `source_page`.
   - Price, campaign, inventory, construction schedule: refer to current store policy.
3. Explain value in the customer's context.
4. Offer a low-risk next step: 到店看样膜、核对车型、看案例、回店复检、确认套餐。
5. Keep the tone helpful, not defensive.

## Objection Patterns

### 太贵了

Emphasize matching needs, not pushing the highest tier.

```markdown
能理解，贴膜不是只看价格，关键是看您最在意的是保护、隔热、隐私还是质保。我们可以先按您的预算做一个合适方案，不一定直接上最高配；再结合车型和使用场景确认有没有必要升级。
```

### 有没有必要贴车衣

Use when the customer worries about scratches, stones, bird droppings, acid rain, water spots, or new-car paint.

```markdown
如果您比较在意原车漆，或者平时高速、露天停车比较多，车衣的价值主要是在漆面保护上，比如减少风沙粉尘、高速石子、酸雨水垢、鸟粪虫尸和轻微剐蹭对漆面的影响。是否贴到高配，要看预算和用车环境。
```

### 会不会伤车

Use the manual's construction process claims.

```markdown
手册里强调的是专车专用电脑裁膜，全程免裁刀、免拆卸操作，目的是从源头降低手工裁切和部件拆卸可能造成的车辆损伤。具体车型建议到店核对施工方案。
```

### 质保多久

```markdown
质保要按具体产品或套餐确认。比如 YM-60 系列车衣手册标注质保期为 3 年，来源页 8。您把客户选的产品发我，我可以帮您查对应来源页。
```

### 贴完能不能洗车/跑高速

```markdown
施工后前三天要注意保护膜面。手册写明车衣与改色施工后三天内不要高速行驶，时速控制在 60 码以内；改色膜三天回店复检，三天内不能洗车。来源页 30。
```

## Guardrails

- Never argue with the customer about price.
- Never promise free rework, compensation, or warranty coverage unless store policy confirms it.
- For quality disputes, advise inspection at the store before giving a conclusion.
