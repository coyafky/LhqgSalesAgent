# Skill Collaboration Flow

Use this flow to choose which sales skill should answer a customer question.

## Flow

1. If the customer's signal is vague or the sales question is leading:
   - Use `ask-with-mom-test`.
2. If the customer has a real pain but the need is not developed:
   - Use `diagnose-with-spin-selling`.
3. If the customer asks an uncomfortable or trust-building FAQ:
   - Use `answer-customer-faq-transparently`.
4. If the customer is ready for a product direction:
   - Use `recommend-film-product`.
5. If the customer resists price, value, warranty, necessity, or risk:
   - Use `handle-film-objections`.
6. If the customer starts bargaining, hesitating, comparing competitors, or delaying through a decision process:
   - Use `negotiate-with-tactical-empathy`.
7. If the sales person needs a customer-facing message:
   - Use `write-sales-followup`.

## Examples

| Query | Primary Skill | Why |
|---|---|---|
| 客户只是说想了解一下，怎么问 | `ask-with-mom-test` | Need is vague; avoid leading questions |
| 新车怕剐蹭，怎么挖需求 | `diagnose-with-spin-selling` | Real pain exists; develop implications and payoff |
| 为什么车衣这么贵 | `answer-customer-faq-transparently` | Trust-building price FAQ |
| YM-80 和 YM-100 怎么选 | `recommend-film-product` | Product comparison with specs |
| 别家便宜很多 | `handle-film-objections` | Price objection |
| 再便宜点就贴 | `negotiate-with-tactical-empathy` | Weak commitment and bargaining |
| 帮我发给客户 | `write-sales-followup` | Customer-facing wording |

## Answer Quality Checklist

- Did we answer the hard question directly?
- Did we avoid inventing price, policy, reviews, and inventory?
- Did we include fit and non-fit guidance?
- Did we cite product source pages when using specs or warranty?
- Did we end with a useful next step?
