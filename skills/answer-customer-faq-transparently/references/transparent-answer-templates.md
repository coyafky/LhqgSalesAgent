# Transparent Answer Templates

Adapt these templates to customer context. Do not invent prices, reviews, or policy.

## Price Difference

```markdown
车衣价格差异通常不只是“膜贵不贵”，还和基材、厚度、涂层、胶水、施工版型、质保和售后有关。基础款适合想做原车漆基础保护、预算稳健的客户；高阶款更适合新车、高端车、经常高速或露天停车、希望质保更长的客户。具体价格和活动要按门店当天政策确认，我可以先按您的车型和预算做一版够用款/升级款对比。
```

## Will Film Damage The Car

```markdown
这个问题可以直接讲清楚：正规的施工重点是降低动刀和拆卸带来的风险。手册里强调有膜有漾采用专车专用电脑裁膜，全程免裁刀、免拆卸操作，用来降低手工裁切和部件拆卸可能造成车辆损伤的风险。具体到您的车型，建议到店核对施工方案和版型。
```

## Car Film Category Choice

```markdown
三类膜解决的问题不一样：车衣主要保护原车漆，适合担心剐蹭、高速石子、鸟粪树胶、酸雨水垢的客户；窗膜主要解决隔热、防晒、防爆、隐私和视线舒适；改色膜主要解决外观风格和原车颜色保护。您如果只想车内不热，先看窗膜；如果新车怕伤漆，优先看车衣；如果想换风格，再看改色膜。
```

## Competitor Is Cheaper

```markdown
能理解您会对比价格。建议不要只看一口价，可以一起看材料、施工方式、是否专车专用版型、质保年限、售后处理和门店稳定性。如果客户只是短期过渡，低预算方案也可能够用；如果是新车、高端车或长期使用，更建议把施工和后续保障一起算进去。我们可以按您的预算做一版不盲目升级的方案。
```

## Warranty

```markdown
质保要按具体产品确认，不同系列不一样。比如 YM-60 系列车衣手册标注质保期为 3 年，来源页 8。您把客户看的具体系列或套餐发我，我可以帮您查对应参数和来源页。
```

## Aftercare

```markdown
施工后前三天要注意保护膜面。手册写明车衣与改色施工后三天内不要高速行驶，时速控制在 60 码以内；改色膜三天回店复检，三天内不能洗车。鸟粪、树脂、胶水、泡沫、地下车库滴水、油烟等也建议及时清理或避免长时间接触膜面。来源页 30。
```

## Not Fit / No Need To Upgrade

```markdown
不一定所有客户都要上最高档。如果客户预算有限、只是基础代步、主要想做基础防护，可以先选够用方案；如果客户经常高速、露天停车、新车原漆保护诉求强，或希望更长质保，再考虑升级更合适。
```
