# Xiaomi Modification Response Patterns

## Learning Logic

For new sales, explain Xiaomi modification products in this order:

1. **车型**：SU7 or YU7.
2. **客户想改变什么**：外观姿态、内饰质感、触感包覆、局部点缀.
3. **产品部位**：front/rear bumper, side skirt, hood, spoiler, mirror, dashboard, steering wheel, door sill, vent, seat back.
4. **销售下一步**：confirm model/year, installation method, stock, quote, warranty, and compliance.

## Category Language

### 外观姿态

Use for 前包围、后包围、侧裙、机盖、尾翼、后视镜、风道口、扰流板、鸭尾.

Customer-facing wording:

> 这类产品主要改变车身外观姿态，适合客户想让车更运动、更有个性。建议先确认想改哪个部位，再看安装方式、预算和门店施工方案。

### 内饰质感

Use for 方向盘、中控面板、出风口、门饰条、迎宾踏板、座椅背板、仪表台饰板.

Customer-facing wording:

> 内饰件更偏向座舱质感升级，客户每天开车都会看到和摸到。可以从方向盘、中控、迎宾踏板这类高频位置开始看。

### 包覆升级

Use for 座椅、顶棚、ABC 柱、门板、中控仪表台、扶手箱.

Customer-facing wording:

> 包覆件主要改变触感和座舱氛围，属于更深度的内饰定制。需要提前确认材质、颜色、施工周期和客户能接受的改动范围。

## Boundaries

Never say:

- 小米官方件
- 原厂件
- 一定合法不用备案
- 价格现在就是多少
- 一定有现货
- 质保多少年 unless a verified policy source exists

Prefer:

- 资料来自内部款式表/供应商图册
- 适配、库存、价格、安装工时、质保以门店确认结果为准
- 外观改装涉及备案或年检时，需要按当地要求确认
