# Skill Collaboration Flow

This skill is an expression enhancer, not the source of product truth.

## Recommended Sequence

1. `ask-with-mom-test`
   - Use when customer signal is vague or the sales question may be leading.
2. `diagnose-with-spin-selling`
   - Use when a real problem exists but the need needs depth.
3. `answer-customer-faq-transparently`
   - Use when the customer asks about price, risk, comparison, reviews, or best fit.
4. `recommend-film-product`
   - Use when the customer needs a specific product or package direction.
5. `handle-film-objections`
   - Use when logic for objection handling is needed.
6. `negotiate-with-tactical-empathy`
   - Use when resistance turns into bargaining, hesitation, competitor comparison, or decision-process negotiation.
7. `strengthen-sales-wording-with-influence`
   - Use to improve tone, trust, and next-step momentum after the substance is correct.
8. `write-sales-followup`
   - Use to produce the final message if needed.

## Example Flow

Customer says:

```text
太贵了，别家便宜很多。
```

First:

- Use `handle-film-objections` for objection logic.
- Use `answer-customer-faq-transparently` if the customer needs education about price factors.
- Use `negotiate-with-tactical-empathy` if the customer is bargaining or emotionally resistant.
- Use this skill to polish wording with reciprocation, authority, and a small commitment.

Final direction:

```markdown
能理解您会对比价格。建议不只看一口价，也看材料、施工方式、版型、质保和售后。我可以先按您的预算做一版够用方案，再给一版升级方案对比，您看完再判断有没有必要升级。具体价格以门店当天政策确认为准。
```

## Quality Checklist

- Is the claim true and verifiable?
- Did we avoid fake scarcity and fake social proof?
- Did we respect the customer's budget and hesitation?
- Did we include one clear, low-pressure next step?
- Did we preserve product facts and price boundaries?
