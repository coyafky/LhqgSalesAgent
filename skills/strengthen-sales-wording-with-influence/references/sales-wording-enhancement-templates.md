# Sales Wording Enhancement Templates

Use these templates as starting points. Replace placeholders with verified details.

## Price Concern

```markdown
能理解您会对比价格。车膜价格差异通常不只是膜本身，还包括基材、厚度、涂层、施工版型、质保和售后。我们可以不直接上最高档，先按您的车型和预算做一版够用方案，再给您一个升级方案对比。具体价格和活动我按门店当天政策确认，避免给您说错。
```

Principles: Reciprocation, Authority, Commitment.

## Invite Store Visit

```markdown
我建议您到店看一下样膜和同类车型案例，这样比只看参数更直观。您不用当场决定，我们先把车型、预算和主要需求确认清楚，再判断哪一档最合适。
```

Principles: Reciprocation, Commitment, Unity.

## Competitor Comparison

```markdown
别家的价格您当然可以参考。建议一起看几个维度：材料、施工方式、是否专车专用版型、质保年限、售后处理和门店稳定性。只看一口价容易漏掉后续使用成本。我们可以按您的预算做一个不盲目升级的对比方案。
```

Principles: Authority, Unity.

## Warranty / Construction Risk

```markdown
这个问题可以直接讲清楚：正规的施工重点是降低动刀和拆卸带来的风险。手册里强调专车专用电脑裁膜，全程免裁刀、免拆卸操作。具体到您的车型，建议到店核对施工版型和方案，质保则按具体产品来源页确认。
```

Principles: Authority, transparency.

## Hesitation Follow-up

```markdown
您先不用急着定。我帮您把选择简化一下：如果主要是保护原车漆，我们看车衣；如果主要是隔热和隐私，我们看窗膜；如果想换外观风格，再看改色膜。您方便先发一下车型和最在意的一个需求吗？我给您做一版更准确的方案。
```

Principles: Reciprocation, Commitment, Liking.

## Too Pushy Rewrite

Bad:

```markdown
今天不定后面就没有这个优惠了，很多客户都选这个，您赶紧定。
```

Better:

```markdown
我先帮您确认门店当天价格和排期，避免给您说错。您如果方便，可以先把车型和预算发我，我按基础款和升级款各做一版，您看完再决定。
```

Principles: Scarcity boundary, Commitment.
