## 完整业务流程示例

```
① 线索录入
   吕佳豪，吕先生问界M8电动踏板抖音来的17628889999
   → 线索管理表

② 跟进记录
   给吕先生写跟进，后天到店看电动踏板
   → 跟进记录表 + 同步更新线索的预计到店时间

③ 报价成交
   报价 MOD-EP-001 1980元，客户同意了
   → 报价表 + 线索状态改为已转商机

④ 客户档案
   自动建档（报价成交后）
   → 客户车辆档案表

⑤ 下工单
   安排梁增炯施工
   → 工单表（待施工）

⑥ 完工
   吕先生完工了
   → 工单表（已关闭）

【自动】第3天 → 客户回访提醒Hook → 提醒销售回访
【自动】第7天 → 客户回访提醒Hook → 提醒销售回访
【自动】第30天 → 客户回访提醒Hook → 提醒保养邀约
```
