### 分析判断

问界M9（50-60万级旗舰SUV），抖音来的→自行搜索过，不是被动推荐。
仅问了电动踏板→还没有全面了解我们的业务。
0次跟进→首次联系，这是第一印象窗口。
电动车踏板作为轻改装入口→客单价不高，但车主消费力强。

### 升单方向

1. **电动踏板（入口）** 确认款式偏好 → 建立联系
2. **隐形车衣** M9尺寸大漆面多，剐蹭成本高
3. **窗膜** M9大天窗+大侧窗，隔热刚需
4. **套餐联动** 踏板+车衣同一天施工，省时间


## 关键要点

1. 入口产品（电动踏板）是信任建立工具 → 不要一上来就强推贵的
2. M9车主的典型顾虑是「伤原车漆」 → 车衣针对性解决
3. 「一次搞定」比「分开两次」更有说服力 → 强调施工效率
4. 电话17620770627 → 微信可加 → 先沟通再看要不要查重
5. 话术中不编造价格、套餐、优惠 → 留门店确认
6. **归属判断是关键第一步**：同名客户必须通过销售负责人字段过滤，不能只看客户名就以为找到了正确的记录
