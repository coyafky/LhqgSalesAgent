# 📋 客户建档 — 销售一句话指南

> 客户成交了，说一句话就能建好客户车辆档案

## 🎯 一句话公式

**「建档/归档 + 客户名 + 什么车 + 颜色 + 状态」**

```
给王先生建档，问界M9珍珠白，刚贴完车衣
```

## 📝 真实示例

| 销售说 | 解析结果 |
|--------|----------|
| *给王先生建档，问界M9珍珠白，2025年上牌，家庭用车* | 客户:王先生 / 车辆:问界M9 / 颜色:珍珠白 / 状态:合作客户 |
| *归档王先生，做了青龙YM-65车衣，关注后续养护* | 客户:王先生 / 一句话:做了车衣关注养护 / 状态:合作客户 |
| *记录一下王先生的车，问界M9，VIN:LSVAU2A36N2100001* | 客户:王先生 / 车辆:问界M9 / VIN:LSVAU2A36N2100001 |
| *王先生是老客户了，做过改色和踏板，下周约了内饰* | 客户:王先生 / 状态:忠诚客户 / 一句话:做过改色踏板… |

## 🔄 客户状态速查

| 你说 | 状态 |
|------|------|
| 刚成交/新客户/刚做完 | → 合作客户 |
| 老客户/做了多次/忠实粉丝 | → 忠诚客户 |
| 还没定/还在考虑 | → 潜在客户 |
| 不来了/流失了/去了别家 | → 流失客户 |

## 🔄 客户标签速查

| 你说 | 标签 |
|------|------|
| 重要客户/大客户/高价值/老板 | → 重要客户 |
| 一般/普通/常规客户 | → 普通客户 |
| 潜在/还没做/还在考虑 | → 潜在客户 |

## 📌 要点

1. **先成交再建档** — 有报价成交后才能建档（建档时会关联报价）
2. **车况说清楚** — 什么车、什么颜色、哪年上牌，越详细越好
3. **一句话进展** — 记录当前状态方便后续维护
4. **VIN码有就填** — 方便以后查维修保养记录

## 🏁 完整流程回顾

```
① 录线索  →  ② 记跟进  →  ③ 报价  →  ④ 成交  →  ⑤ 建档 ✅
```
