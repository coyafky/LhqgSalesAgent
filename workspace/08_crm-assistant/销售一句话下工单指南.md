# 📋 下工单 — 销售一句话指南

> 客户到店施工了，说一句话就能下好工单

## 🎯 一句话公式

**「下工单/派工 + 客户名 + 做什么 + 谁做 + 什么时间」**

```
王先生到店了，下工单，青龙YM-65，梁增炯做
```

## 📝 真实示例

### 创建工单

| 销售说 | 解析结果 |
|--------|----------|
| *王先生到店了，下个工单，青龙YM-65，梁增炯做* | 客户:王先生 / 项目:青龙YM-65 / 师傅:梁增炯 / 状态:待安排 |
| *派工，王先生明天上午来，白虎YM-70，阮其梓施工* | 客户:王先生 / 项目:白虎YM-70 / 师傅:阮其梓 / 时间:明天 / 状态:待施工 |
| *王先生到店了，贴青龙YM-65，梁师傅做，今天下午2点* | 客户:王先生 / 项目:青龙YM-65 / 师傅:梁增炯 / 时间:今天下午2点 |

### 更新工单状态

| 销售说 | 解析结果 |
|--------|----------|
| *王先生开始施工了* | 工单状态 → 施工中 |
| *王先生贴完了，验收一下* | 工单状态 → 待验收 |
| *王先生验收通过，没问题* | 工单状态 → 待交付 |
| *王先生提车走了，完工* | 工单状态 → 已关闭 |

## 🔄 工单状态速查

| 你说 | 状态 | 说明 |
|------|------|------|
| 新工单/下个单/派工 | → 待安排 | 刚创建 |
| 安排了/明天做/排期了 | → 待施工 | 已安排师傅和时间 |
| 开始施工/在贴了/正在做 | → 施工中 | 师傅正在干活 |
| 做完了/贴好了/验收 | → 待验收 | 等质检 |
| 验收过了/没问题/OK | → 待交付 | 等客户提车 |
| 交付了/提车了/开走了/完工 | → 已关闭 | 工单结束 |

## 🔄 施工师傅速查

| 你说 | 师傅 |
|------|------|
| 梁增炯/梁师傅/阿炯 | → 梁增炯（中工，车衣/窗膜/改色） |
| 阮其梓/阮师傅/阿梓 | → 阮其梓（中工，车衣/窗膜/改色） |

## 📌 要点

1. **先建档再下工单** — 必须先有客户档案和成交报价
2. **师傅说清楚** — 梁增炯还是阮其梓，不说的话助理会问
3. **时间要预约** — 说清楚什么时候施工
4. **完工要说** — 客户提车后记得说"完工了"关闭工单

## 🏁 全流程回顾

```
① 录线索 → ② 记跟进 → ③ 报价 → ④ 成交 → ⑤ 建档 → ⑥ 下工单 → ⑦ 完工 ✅
```
