# CRM 数据质量规则

## 必填完整度

一条有效销售线索至少应包含：

- 客户称呼或姓名。
- 联系方式或可识别微信备注。
- 车型。
- 产品意向。
- 当前阶段。
- 下一步动作。
- 下次跟进时间。
- 负责人。

## 重复检查

写入前应按以下顺序查重：

1. 手机号。
2. 微信备注。
3. 客户称呼 + 车型。
4. 客户称呼 + 产品意向 + 最近创建时间。

## 阶段一致性

- 已成交：必须有成交产品或施工记录。
- 已报价：应有报价方案或备注，但不要求 agent 编造价格。
- 到店邀约：必须有到店时间或待确认时间。
- 流失：必须记录流失原因。

## 跟进时间

- “明天”“周五”“周末”必须解析到具体日期。
- 无法确定具体日期时，先追问。
- 过去时间不能作为下次跟进时间。

## 隐私与合规

- 手机号、微信号、真实客户身份不写入 workspace 文档。
- 只在 CRM 表中保存客户隐私。
- 对外话术不得泄露内部阶段、利润、折扣权限或销售备注。
