# 📋 售后 — 销售一句话指南

> 客户反馈问题或做回访时，说一句话就能记录售后单

## 🎯 一句话公式

**「客户名 + 什么问题 + 售后类型」**

```
王先生车衣边缘翘了，安排返工
```

## 📝 真实示例

### 客户反馈问题

| 销售说 | 解析结果 |
|--------|----------|
| *王先生车衣边缘翘了，安排返工* | 类型:返工 / 问题:边缘翘 / 状态:待处理 |
| *王先生车衣有点发黄，走质保* | 类型:质保 / 问题:发黄 / 状态:待处理 |
| *王先生投诉施工有划痕，不太满意* | 类型:投诉 / 问题:划痕 / 状态:待处理 |

### 售后回访/保养

| 销售说 | 解析结果 |
|--------|----------|
| *回访王先生，他说车衣很好，很满意* | 类型:复购回访 / 满意度:满意 / 状态:已解决 |
| *王先生约了周六回来做保养* | 类型:保养 / 预约:周六 / 状态:待处理 |

### 更新售后进度

| 销售说 | 解析结果 |
|--------|----------|
| *王先生的返工处理好了，客户满意* | 状态→已解决 / 满意度:满意 |
| *王先生的问题解决了，热风收边处理* | 状态→已解决 / 方案:热风收边 |

## 🔄 售后类型速查

| 客户反馈 | 类型 |
|---------|------|
| 起泡/翘边/脱胶/开裂/划痕 | → 返工 |
| 发黄/变色/质保期内 | → 质保 |
| 态度不好/不满意/投诉 | → 投诉 |
| 问问/咨询一下 | → 咨询 |
| 约保养/护理/养护 | → 保养 |
| 回访/问满意度/推荐朋友 | → 复购回访 |

## 🔄 售后状态速查

| 你说 | 状态 |
|------|------|
| 新反馈/还没处理 | → 待处理 |
| 在处理/师傅在看 | → 处理中 |
| 修好了/解决了 | → 已解决 |
| 不用处理/客户说没事 | → 已关闭 |

## 🔄 满意度速查

| 客户说 | 满意度 |
|--------|--------|
| 很好/满意/没问题 | → 满意 |
| 还行/一般/就这样 | → 一般 |
| 不行/不好/不满意 | → 不满意 |

## 📌 要点

1. **先有工单再有售后** — 售后要关联原工单
2. **类型说清楚** — 返工还是质保？直接影响处理流程
3. **问题具体说** — "边缘翘了2cm"比"有点问题"更有用
4. **处理完要反馈** — 修好了记一笔客户满意度
