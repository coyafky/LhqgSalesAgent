# 📋 销售一句话录线索指南

> 不用打开表格找字段，直接跟销售助理说一句话就能录客户

## 🎯 一句话公式

**「负责人 + 客户称呼 + 车型 + 意向产品 + 来源 + 电话」**

```
吕佳豪  王先生  问界M9  电动踏板  抖音来的  17620770627
```

## 📝 真实示例

### 新增线索

| 销售说 | 助理解析 |
|--------|----------|
| *吕佳豪，王先生问界M9电动踏板抖音来的17620770627* | 负责人:吕佳豪 客户:王先生 车型:SUV 产品:轻改装 来源:抖音 电话:17620770627 |
| *刘志培，李姐特斯拉Model Y，想贴车衣怕石子，小红书来的18512345678* | 负责人:刘志培 客户:李姐 车型:SUV 产品:隐形车衣 来源:小红书 |
| *张秋苑，陈总理想L9窗膜，美团来的13988889999，约了周六到店* | 负责人:张秋苑 车型:SUV 产品:窗膜 来源:美团点评 到店时间:周六 |
| *帮我录一个，王姐，宝马X5改色，抖音来的，加微信了，电话没给* | 客户:王姐 车型:SUV 来源:抖音 → 助理会追问电话 |

### 常用说辞速查

| 你说 | 助理理解成 |
|------|-----------|
| 车衣 / 隐形车衣 | → 隐形车衣 |
| 太阳膜 / 窗膜 / 隔热膜 | → 窗膜 |
| 改色 / 换色 / 贴个颜色 | → 改色膜 |
| 电动踏板 / 踏板 / 轻改 | → 轻改装 |
| 抖音 / 小红书 / 转介绍 | → 对应来源渠道 |

## 📌 要点

1. **负责人说清楚** — 吕佳豪 / 刘志培 / 张秋苑，不然默认未分配
2. **电话最优先** — 有电话写最前面，没有电话助理会追问
3. **产品说人话** — 说"电动踏板"就行，助理会自动归类为轻改装
4. **车型报全称** — 问界M9、理想L7、Model Y 都行，助理只填SUV/轿车/MPV
5. **不用等** — 录完助理会让你确认，说"确认写入"就完成了
