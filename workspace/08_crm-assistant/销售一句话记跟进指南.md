# 📋 记跟进 — 销售一句话指南

> 跟客户聊完后，说一句话就能记下沟通情况

## 🎯 一句话公式

**「客户名 + 聊了什么 + 结果怎么样 + 下次什么时候」**

```
王先生 今天微信聊了 说价格有点高要和老婆商量 后天再问
```

## 📝 真实示例

| 销售说 | 解析结果 |
|--------|----------|
| *王先生今天微信聊了，说价格有点高，要和老婆商量，后天再问* | 跟进方式:微信沟通 / 结果:需再次跟进 / 下次:后天 |
| *刚跟李姐通了电话，约了周六下午到店看膜* | 跟进方式:电话沟通 / 结果:已预约到店 / 到店:周六 |
| *陈总微信发了报价，说比别家贵，要再想想* | 跟进方式:微信沟通 / 结果:需再次跟进 / 异议:价格 |
| *张先生没接电话，明天再打试试* | 跟进方式:电话沟通 / 结果:待回复 / 下次:明天 |
| *王姐今天到店了，直接定了车衣+窗膜，交了定金* | 跟进方式:线下拜访 / 结果:已成交 |
| *记录一下赵先生，他朋友介绍来的，问了改色价格，发案例了* | 跟进方式:微信沟通 / 结果:已联系 / 下次:待追问 |

## 🔄 跟进结果速查

| 你说 | 助理解析 |
|------|---------|
| 聊了/联系上了/电话通了 | → 已联系 |
| 没回/没接/等回复 | → 待回复 |
| 约了周六/说周末来看/约定到店 | → 已预约到店 |
| 报了价/发了报价/问了价格 | → 已报价 |
| 定了/付款了/施工了/成交了 | → 已成交 |
| 不考虑/不需要/暂时不 | → 暂无意向 |
| 考虑中/要和家人商量/再想想/比比价 | → 需再次跟进 |

## 🔄 跟进方式速查

| 你说 | 助理解析 |
|------|---------|
| 微信/微聊/发了微信/发案例 | → 微信沟通 |
| 电话/打电话/电联/通了电话 | → 电话沟通 |
| 到店/见面/拜访/来店里了 | → 线下拜访 |

## 📌 要点

1. **先说客户名** — 助理会根据名字搜索对应线索
2. **结果说清楚** — 定了还是再想想，直接影响下一步
3. **下次跟进时间** — 说清楚啥时候再跟，没说的话助理会问
4. **先录线索再跟进** — 如果还没有这个客户的线索，先"录客户"再"记跟进"
