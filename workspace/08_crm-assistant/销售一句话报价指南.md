# 📋 报价 — 销售一句话指南

> 客户有意向了，说一句话就能填好报价单

## 🎯 一句话公式

**「客户名 + 产品 + 价格 + 状态」**

```
王先生，青龙YM-65，报9800
```

## 🔄 报价状态速查

| 你说 | 状态 |
|------|------|
| 报个价/草稿/先写一下 | → 草稿 |
| 发给客户了/报价已发/发过去了 | → 已发客户 |
| 在谈/客户在考虑/跟进中 | → 跟进中 |
| 定了/成交了/客户同意/付款了 | → 已成交 |
| 不要了/过期/作废 | → 已作废 |

## 📌 要点

1. **先说客户名** — 助理自动搜索对应线索
2. **产品名说准确** — "青龙YM-65"比"那个六千五的车衣"更准
3. **先有线索再报价** — 客户必须先录了线索，才能报价
4. **报价→跟进一条龙** — 报完价记得"记跟进"记录客户反应
5. **价格数字说清楚** — 说"九千八"或"9800"都可以
