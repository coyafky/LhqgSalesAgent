# 自然语言填写示例

## 新增线索

销售输入：

```text
新增一个客户，陈先生，问 Model Y 车衣，主要怕高速石子，预算别太高，抖音来的，明天下午回访。
```

Agent 识别：

```markdown
操作类型：新增
客户称呼：陈先生
车型：Model Y
来源渠道：抖音
产品意向：隐形车衣
核心诉求：高速石子、漆面保护、预算敏感
当前阶段：新线索
下一步动作：明天下午回访

缺失：联系方式、负责人
```

## 更新跟进

销售输入：

```text
更新陈先生，他说价格有点高，要和老婆商量，先发两个中端车衣案例给他，周五上午再问。
```

Agent 识别：

```markdown
操作类型：更新
客户：陈先生
客户异议：价格异议、家人决策
下一步动作：发送中端车衣案例
下次跟进时间：周五上午
备注：客户认为价格偏高，需要和家人商量。
```

## 售后记录

销售输入：

```text
王女士理想 L8 改色膜今天交车了，提醒她三天不要洗车，三天回店复检。
```

Agent 识别：

```markdown
操作类型：更新
客户：王女士
车型：理想 L8
产品意向：改色膜
当前阶段：已交车/售后提醒
下一步动作：三天回店复检
备注：已提醒三天内不要洗车。
```
