# Roadmap

## Phase 1: 销售工作台成型

- 建立项目级 PRD 与 SDD。
- 梳理销售场景 SOP。
- 对齐现有 skills 与真实销售流程。
- 明确价格、政策、售后、改色生图边界。

## Phase 2: 知识库质量提升

- 复核车衣和窗膜参数来源页。
- 补录改色膜 p.26-p.29 的产品细项。
- 建立车型、产品、FAQ、话术的统一索引。
- 为高频问题补充验收用例。

## Phase 3: 一线反馈闭环

- 建立销售反馈表。
- 每周归档高频问题和失效话术。
- 将有效话术沉淀为 SOP 或 skill reference。
- 对失败案例做回归测试。

## Phase 4: 交付体验优化

- 优化飞书入口词和返回格式。
- 改色图交付增加色卡来源、比例校验和免责声明。
- 建立门店政策的时效性管理方式。
