# 轮毂参数与安全边界

轮毂是安全件。Hermes Agent 可以帮助销售解释概念、提醒风险和收集信息，但不能替门店、供应商或技师做最终适配承诺。

## 必须确认的 8 个信息

| 信息 | 为什么必须确认 |
| --- | --- |
| 车型 / 年份 / 版本 | 同一车型不同年款可能轮毂参数不同 |
| 当前轮胎规格 | 从 `225/45R18` 这类数字可判断轮毂直径 |
| 轮毂尺寸 | 尺寸变化可能需要换胎，并影响舒适和年检 |
| J 值 | 决定轮胎宽度匹配 |
| ET 值 | 决定内外位置，影响蹭刹车、蹭悬挂、方向盘手感 |
| PCD | 孔距不匹配通常不能直装 |
| 中心孔 | 大于轴头需变径圈，小于轴头装不上 |
| 刹车避让 / 载荷等级 | 改过大卡钳、新能源车和重车必须重点确认 |

## ET 值怎么谨慎说

原始学习资料给了一个销售经验：ET 改动建议控制在原厂附近，不要大幅偏离。对外表达时建议更克制：

> ET 值影响轮毂外凸或内缩。想更齐边可以适当调整，但必须结合原厂 ET、J 值、轮胎宽度、悬挂和翼子板空间确认。改太多可能蹭翼子板、蹭悬挂，也可能影响方向盘手感和刹车稳定。

**不要这样说：**
- “ET 改多少都可以。”
- “只要好看就行。”
- “绝对不会蹭。”
- “不用看刹车和悬挂。”

## 变径圈怎么说

> 如果新轮毂中心孔大于原厂轴头，通常需要变径圈来居中；如果中心孔小于轴头，就装不上。是否需要变径圈，要看实测参数和产品规格。

**风险提醒：** 不合适的中心孔或缺失变径圈，可能导致高速抖动、动平衡异常甚至安全风险。

## 新能源车轮毂边界

新能源车通常更重、瞬时扭矩更大，轮毂选择要更关注：

- 载荷等级是否满足原车要求。
- 是否有 EV Compatible 或电动车适配案例。
- 大尺寸和低扁平比可能影响舒适性、爆胎风险和续航。
- 封闭式或 Aero 轮毂更有利于风阻，开放式轮毂通常更好看但可能影响续航。

**对客话术：**
> 电车改轮毂不是不能改，而是不能随便用油车思路改。车重、扭矩、续航和载荷都要一起看。我们先确认原厂参数和用车场景，再判断适合走颜值、轻量化还是续航优先。

## 年检与备案边界

各地政策和实际检测要求可能不同。销售不要替客户承诺“肯定过”或“肯定不用备案”。

**稳妥表达：**
> 轮毂尺寸、外观和轮胎规格是否影响年检或是否需要备案，要以当地车管和检测要求为准。我们可以先帮您整理参数，再让门店或当地机构确认。

## 投诉排查框架

客户改完轮毂后说方向盘抖、刹车抖、跑偏，可按以下顺序排查：

1. 是否做了动平衡。
2. 中心孔是否匹配，是否需要变径圈。
3. PCD 和螺栓规格是否正确。
4. ET 值是否偏离太多。
5. 轮胎是否失圆、鼓包或安装不良。
6. 轮毂是否变形、修复或翻新。
7. 是否改过刹车、避震、法兰盘、转接盘。
