## 可用于对客表达的内容

| 内容 | 可否对客说 | 边界 |
| --- | --- | --- |
| 轮毂基础术语：尺寸、J 值、ET、PCD、中心孔 | 可以 | 作为概念解释，不替客户做最终适配 |
| 制造工艺：铸造、旋压、锻造 | 可以 | 不把某工艺说成所有场景绝对最好 |
| 轮毂是安全件，低价无标需谨慎 | 可以 | 不攻击具体商家或未核验品牌 |
| 新能源车需关注载荷、扭矩、续航、EV 兼容 | 可以 | 不承诺具体续航损失或适配结果 |
| 轮毂与车衣/改色/太阳膜的交叉销售逻辑 | 可以 | 仅作为销售启发，不硬推 |

## 必须确认后才能说的内容

| 内容 | 为什么需确认 |
| --- | --- |
| 某车型具体 PCD、中心孔、ET、J 值 | 同车型不同年款/版本可能不同 |
| 某套轮毂能否直装 | 需确认参数、刹车避让、载荷、轮胎匹配 |
| 品牌真伪和认证 | 需品牌官方、授权渠道或专业门店核验 |
| 年检、备案、合法性 | 受当地政策和实际检测影响 |
| 价格、库存、质保、安装工时 | 需要门店或供应商实时确认 |
| 续航损失百分比、重量差、性能提升 | 受轮毂重量、轮胎、车型、路况影响 |

## Agent 回答规则

1. 可以解释轮毂知识，但不能替代技师和供应商适配判断。
2. 客户问“能不能装”，先收集车型、年份、轮胎规格、原厂参数和改装目标。
3. 客户问“会不会影响年检/质保”，必须提醒以当地政策、4S/厂家和门店确认为准。
4. 客户问“这套是不是正品”，只能提供核验思路，不能未经验证下结论。
5. 客户问“网上低价能不能买”，强调轮毂是安全件，建议选择有品牌、有认证、有售后的渠道。
