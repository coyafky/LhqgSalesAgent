# 汽车轮毂 学习路径

## 阶段 1：扫盲（1-2 小时）
**目标**：能看懂轮毂参数，知道自己的车能装什么

- [ ] 读完 [术语表](01-terminology.md) Level 1（10 个必知术语）
- [ ] 理解「尺寸 + J 值 + ET 值 + PCD + 中心孔」这个五参数体系
- [ ] 用自己车的参数去查一下，看看原厂轮毂的数据
- [ ] 能区分「锻造 > 旋压 > 铸造」三种工艺

**检验**：随便打开一个轮毂商品页，能看懂参数表。

---

## 阶段 2：深入（半天）
**目标**：能跟改装店老板对话，不会被人忽悠

- [ ] 读完 [术语表](01-terminology.md) Level 2（9 个常用术语）
- [ ] 深入理解 [核心概念](02-core-concepts.md) 全部 5 个概念
- [ ] 浏览 [关键玩家](03-key-players.md)，记住国际 5 大品牌 + 国产 3 大品牌
- [ ] 读完 [常见问题](04-common-questions.md) 全部 18 个问题
- [ ] 找 3 个不同价位的轮毂方案，对比参数和价格

**检验**：能为朋友的车推荐一套轮毂方案，并解释为什么这样选。

---

## 阶段 3：实战（1-2 天）
**目标**：能独立挑选并购买一套适合自己的轮毂

- [ ] 确定自己的车型参数（查说明书或原厂轮毂钢印）
- [ ] 确定预算和需求（走街？赛道？姿态？）
- [ ] 筛选 3-5 个候选方案
  - 品牌 + 型号
  - 尺寸 + J 值 + ET 值
  - 工艺（铸造/旋压/锻造）
  - 价格 + 购买渠道
- [ ] 对比分析：哪个性价比最高？哪个最轻？哪个最好看？
- [ ] 最终选定一套，去改装店/线上找报价

**检验**：下单买回来装上不后悔。

---
