# 轮毂设计风格 & 车型搭配指南

> 客户问「我车装这套好不好看？」——你的审美框架。

---

## 七种轮毂造型分类

### 1. 五辐式（5-Spoke）— 永不过时

```
    ★
   ╱ ╲
  ╱   ╲
 ╱     ╲
★       ★
```

| 特点 | 硬朗、简洁、运动感强 |
|------|---------------------|
| 代表 | RAYS TE37, BBS CH-R, ENKEI RPF1 |
| 适合车型 | 轿跑、跑车、运动轿车 |
| 不适合 | 豪华商务车（太运动，不搭） |
| 视觉效果 | 显大，暴露刹车（需要大卡钳才好看） |

---

### 2. 多辐式（Multi-Spoke）— 优雅豪华

```
    ✦ ✦
   ╱ ╲ ╱ ╲
  ╱ ╲ ╱ ╲ ╱ ╲
 ╱ ╲ ╱ ╲ ╱ ╲ ╱ ╲
✦ ✦ ✦ ✦ ✦ ✦ ✦ ✦
```

| 特点 | 优雅、豪华、视觉上填满轮拱 |
|------|--------------------------|
| 代表 | BBS LM, OZ Superturismo |
| 适合车型 | 豪华轿车、行政车、SUV |
| 不适合 | 小钢炮（辐条太密不运动） |
| 视觉效果 | 显小——同样 18 寸，多辐看起来比五辐小一圈 |

---

### 3. Y 辐式（Y-Spoke）— 现代运动

```
    \│/
     ★
    ╱ ╲
   ╱   ╲
```

| 特点 | 像字母 Y 分叉，现代感强、动感 |
|------|------------------------------|
| 代表 | VOSSEN VFS, WORK Emotion, 雅泛迪 MR2 |
| 适合车型 | 运动轿车、轿跑 SUV、新能源 |
| 不适合 | 复古风格的车 |
| 视觉效果 | 比五辐更立体，辐条末端分叉有视觉延伸感 |

---

### 4. 网状（Mesh）— 经典复古

```
★★★★★★★★
╲╱╲╱╲╱╲╱╲╱
```

| 特点 | 细密交叉网格，经典复古，欧系味道 |
|------|---------------------------------|
| 代表 | BBS RS / SUPER RS, WORK VS |
| 适合车型 | 老宝马 E30/E36/E46、老奔驰、JDM 复古 |
| 不适合 | 太现代的车（造型冲突） |
| 视觉效果 | 非常有年代感，轮毂本身像一件艺术品 |

---

### 5. 涡轮式（Turbine）— 动感旋转

```
    ╱
   ╱
  ╱  ╲
 ╱    ╲
╱      ╲
```

| 特点 | 像涡轮叶片一样有方向性，视觉动态感强 |
|------|-----------------------------------|
| 代表 | 原厂奥迪/奔驰 AMG 部分款式 |
| 适合车型 | 运动型新能源、轿跑 SUV |
| 不适合 | 商务车（太激进） |
| 注意事项 | 有「左旋」和「右旋」之分——装的左右对不对称=懂不懂细节 |

---

### 6. 大抛边（Deep Lip）— 姿态王者

```
┌─────────┐
│  ╱ ╲  │  ← 辐条内凹
│ ╱   ╲ │
│╱     ╲│
└─────────┘
  ← 抛边（Lip）→
```

| 特点 | 轮辋边缘宽大凸出，辐条向内凹陷 |
|------|-----------------------------|
| 代表 | WORK Meister, BBS LM, SSR Professor |
| 适合车型 | 姿态改装车、低趴风格、跑车 |
| 不适合 | SUV、家用车、原厂高度 |
| 视觉效果 | 非常有厚度感——但必须配合降车身 |

> ⚠️ 大抛边轮毂「显大」但实际 J 值偏小——这是「假 J 值」轮毂的常见坑。

---

### 7. 封闭式/Aero 式（EV 专用）

```
┌─────────┐
│▓▓▓▓▓▓▓▓│  ← 大面积封闭
│▓▓  ▓▓▓▓│
│▓▓▓▓▓▓▓▓│
└─────────┘
```

| 特点 | 辐条之间大面积封闭，减少空气阻力 |
|------|---------------------------------|
| 代表 | 特斯拉原厂 Aero 轮毂盖、蔚来 ET7 原厂 |
| 适合车型 | 新能源车（降低风阻=增加续航） |
| 不适合 | 性能车（影响刹车散热） |
| 视觉效果 | 科技感强，但很多人觉得「丑」→ 特斯拉车主最常见的拆盖改装 |

## 轮毂颜色搭配法则

| 车身颜色 | 推荐轮毂颜色 | 避雷 |
|---------|------------|------|
| 白色 | 枪灰、亮黑、银色 | ❌ 白车轮毂不要配白色 |
| 黑色 | 亮银、古铜、金色 | ❌ 全黑=黑成一团 |
| 灰色/银色 | 枪灰、亮黑、哑光银 | 都比较百搭 |
| 红色 | 亮黑、枪灰、金色 | ❌ 红色车轮毂不要配红色 |
| 蓝色 | 银色、亮黑、古铜 | 比较百搭 |
| 绿色 | 古铜、金色、亮黑 | ❌ 不要配银色（红配绿赛狗屁的变体） |

## 车型 × 风格速配表

| 车型类型 | 首选风格 | 备选 | 避雷 |
|---------|---------|------|------|
| 日系小钢炮（思域/飞度） | 五辐/Y 辐 | 多辐 | 大抛边（太高了不好看） |
| 德系轿车（3系/A4/C级） | 多辐/Y 辐 | 五辐 | 网状（年代感不搭） |
| 豪华 SUV（X5/GLE） | 多辐 | Y 辐 | 五辐（显单薄） |
| 轿跑 SUV（Model Y/ES6） | Y 辐/涡轮 | 五辐 | 网状（太复古） |
| 姿态改装（低趴） | 大抛边/网状 | 三片式定制 | 原厂款（浪费了低趴） |
| 肌肉车（Mustang） | 五辐 | Y 辐 | 细多辐（不搭粗犷感） |
| JDM 复古 | 网状/大抛边 | 五辐 | Y 辐/涡轮（太现代） |

## 尺寸选择的视觉规则

```
车身颜色深 + 轮毂颜色浅 = 轮毂显大
车身颜色浅 + 轮毂颜色深 = 轮毂显小

多辐 → 显小（填满了但每根细）
五辐 → 显大（每根粗，中间空）

抛边厚 → 轮毂显小一寸（抛边挤占了视觉直径）
```

> **黄金法则**：你想让轮毂看起来大 → 选五辐 + 亮色 + 无抛边。

## 销售怎么用这套框架？

客户给你看一台车 → 3 秒判断：

1. 什么车？（日系/德系/新能源？轿车/SUV？）
2. 什么风格？（原厂高度/运动/低趴姿态？）
3. 匹配推荐：「你这台 XX，我建议考虑 Y 辐或者五辐的，枪灰色配你车身最好看。我找个案例给你看。」
