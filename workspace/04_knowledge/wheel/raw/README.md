> 🕐 生成时间：2026-06-08

## 一句话定义

轮毂是连接车轮和车轴的金属部件，支撑整车重量，决定车辆外观、操控性能和安全。

## 为什么重要

- 轮毂是 **簧下质量** 的核心——每减轻 1kg 轮毂 ≈ 车身减重 15kg 的效果
- 直接影响操控：更轻的轮毂 = 更敏捷的转向 + 更短的刹车距离
- 中国改装市场快速增长，轮毂是「买车后第一个想换的零件」
- 新能源车大尺寸轮毂趋势明显（18-22 寸成为主流）
