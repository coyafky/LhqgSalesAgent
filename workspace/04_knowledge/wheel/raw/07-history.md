# 汽车轮毂 发展简史

> 从马车轮到碳纤维——两百年轮毂进化史。

---

## 史前时代：马车轮的遗产（1800 年代以前）

轮毂（Hub）这个词本身就是从马车时代来的。马车轮的中心有一个木质"毂"——辐条汇聚的中心点。车轮绕着这个毂旋转，毂套在轴上。

**马车轮给汽车轮留下了什么？**
- 辐条（Spoke）概念——从木质辐条到钢丝辐条再到铝合金辐条
- 中心孔概念——毂套轴的结构一直沿用至今
- 滚动摩擦原理——轮子越大，过障碍越轻松

---

## 第一代：钢丝轮毂（1880s-1930s）

```
马车轮 → 自行车轮 → 摩托车轮 → 汽车钢丝轮
```

**标志性技术**：Rudge-Whitworth 中央锁止系统（1908 年）

- 汽车刚诞生时，直接借用自行车/摩托车的钢丝轮毂设计
- 用几十根钢丝辐条连接轮辋和中心毂，轻且有一定缓冲能力
- 需要经常调整辐条张力（就像自行车"拿龙"一样）
- 1908 年英国 Rudge-Whitworth 发明了中央锁止螺母——一个螺母就能固定整个轮子，是赛车快拆的前身

**为什么被淘汰？**
- 维护麻烦——几十根辐条要定期调张力
- 无法密封——用内胎的时代还行，无内胎时代就废了
- 刹车散热差——钢丝轮遮挡了刹车系统的空气流通

**但为什么还在用？**
- 复古摩托车、老爷车、部分越野车（钢丝轮在越野时有一定的弹性缓冲）

---

## 第二代：钢制轮毂（1930s-1970s）

```
冲压钢板 → 焊接 → 钢圈
```

**革命性变化**：一体式冲压钢圈

- 用冲压机把钢板压出轮辋和轮辐的形状，然后焊接在一起
- 便宜——量产成本极低
- 结实——钢的韧性好，撞坑不易断裂，最多变形
- 重——钢密度大，一个 15 寸钢圈约 8-10 kg，同尺寸铝合金约 6-7 kg

**钢圈为什么还在大量使用？**
- 全世界绝大多数经济型轿车的备胎还是钢圈
- 商用车（卡车/货车）几乎全是钢圈——载重能力铝合金无法替代
- 2024 年中国钢圈市场依然超过 200 亿元

**但乘用车为什么抛弃了它？**
- 太重——簧下质量大，影响操控和油耗
- 丑——冲压出来的造型有限，盖个塑料罩子遮丑
- 散热差——钢导热性不如铝，连续刹车后热量积聚
- 易生锈——冬天撒盐的路面几年就锈了

---

## 第三代：铝合金铸造轮毂（1958-现在）

```
铝水 → 倒入模具 → 冷却成型 → 机加工 → 成品
```

**里程碑**：1958 年，第一只整体铸造铝合金轮毂诞生。

- 1930 年代德国人已经尝试将铝质轮辋和钢制辐条结合
- 1958 年实现了真正的「一体式铝合金铸造」
- A356 铝合金（92% 铝 + 硅/镁/铁）成为铸造轮毂的标准材料
- 比钢圈轻 30%
- 散热好——铝的导热系数是钢的 3 倍
- 造型自由——铸造可以做出复杂的辐条设计

**1970s-1990s**：铸造轮毂快速普及
- 日本车企大规模使用铝合金轮毂
- 中国 1980 年代末开始生产铝轮毂（秦皇岛戴卡，即后来的中信戴卡）
- 2024 年，全球乘用车铝合金轮毂渗透率超过 80%

---

## 第四代：锻造轮毂（1970s-现在）

```
铝坯 → 万吨压力机锻压 → CNC 精加工 → 成品
```

**锻造 ≠ 铸造**——这是从「倒模具」到「打铁」的质变。

- 1970s BBS 开始为赛车生产锻造轮毂
- 1980s RAYS 投入万吨锻造压力机，生产 TE37（至今仍是改装圈图腾）
- 材料从 A356（铸造铝）升级到 6061-T6（航空铝）
- 比同尺寸铸造轮毂再轻 20-30%，强度高 30%

**锻造的成本壁垒**：
- 万吨压力机价格千万级
- 每个轮毂需要 CNC 精加工（耗时是铸造的 5-10 倍）
- 材料贵——6061 航空铝单价是 A356 的 2-3 倍
- 因此锻造轮毂价格是铸造的 5-10 倍

---

## 第五代：旋压轮毂（1990s-现在）

```
铸造毛坯 → 旋转加压 → 轮辋变薄变密 → 接近锻造性能
```

**日本 Enkei 的 MAT 技术**（Most Advanced Technology）是旋压工艺的鼻祖。

- 在铸造轮毂的基础上，对轮辋部分进行旋转加压
- 金属分子被挤压得更紧密，呈纤维状排列
- 强度接近锻造（约 90%），但成本接近铸造（约 1.5 倍）
- 重量介于铸造和锻造之间——比铸造轻 15-20%

**为什么旋压是「甜点」工艺？**
- 性价比最高——用 1.5 倍铸造价格拿到 90% 的锻造性能
- ENKEI RPF1（旋压）是赛道圈速验证最多的「平民神圈」

---

## 第六代：碳纤维 & 复合材料（2010s-现在）

```
碳纤维预浸料 → 高温高压固化 → 成型
```

- HRE 推出量产碳纤维轮毂（单只 3 万+）
- 比锻造铝合金再轻 40%
- 但价格是锻造的 3-5 倍
- 目前仅用于 Hypercar（科尼赛克/帕加尼）和顶级改装

**碳纤维轮毂的致命伤**：
- 一次撞击就可能内部断裂（不像铝合金可以变形预警）
- 无法修复——撞了就废
- 所以目前不是主流，但代表未来方向

## 年表总结

| 时期 | 技术 | 材料 | 代表 | 重量（18寸/只） |
|------|------|------|------|----------------|
| 1880s-1930s | 钢丝轮 | 钢丝 + 钢毂 | Rudge-Whitworth | 8-10 kg |
| 1930s-1970s | 冲压钢圈 | 钢板 | 各 OEM 原厂 | 10-13 kg |
| 1958-现在 | 铸造铝合金 | A356 | 中信戴卡 | 9-11 kg |
| 1970s-现在 | 锻造铝合金 | 6061-T6 | BBS / RAYS | 7-9 kg |
| 1990s-现在 | 旋压 | 铝合金 | ENKEI MAT | 8-10 kg |
| 2010s-现在 | 碳纤维 | 碳纤维 | HRE Carbon | 5-7 kg |

## 中国轮毂产业简史

```
1988    秦皇岛戴卡成立 → 中国第一个铝轮毂工厂
1990s   跟随汽车产业爆发 → 产量从 0 到全球第一
2000s   WTO 后出口激增 → 中信戴卡成为全球最大铝轮毂制造商
2010s   从铸造到锻造转型 → 国产锻造品牌崛起（雅泛迪/ARTKA）
2024    中国铝轮毂年产能超 2 亿件 → 全球占比 60%+
```
