## Agent 使用建议

- 客户问“J 值、ET、PCD、中心孔、锻造、旋压、铸造是什么”：优先读 `sales-knowledge.md` 和 `raw/01-terminology.md`。
- 客户问“我这车能装什么轮毂”：先收集车型、年份、原厂规格、当前轮胎侧面数字、轮毂背面钢印；再提醒需门店或供应商确认。
- 客户问“ET 能不能改小、轮毂能不能外凸”：优先读 `parameter-and-safety-boundary.md`。
- 客户问“几百块轮毂能不能买、真假怎么辨”：读 `sales-qa.md` 和 `raw/11-authenticity-guide.md`。
- 销售想用轮毂话题带出车膜/PPF/改色：读 `cross-sell-playbook.md`。

## 总边界

- 轮毂是安全件，不能凭经验替客户确定最终适配。
- 任何轮毂改装建议都必须确认车型、年份、轮胎规格、PCD、中心孔、ET、J 值、刹车避让、载荷等级和年检/备案要求。
- 价格、库存、品牌真伪、认证、施工工时、备案政策、质保和售后必须以门店、供应商、品牌官方或当地法规为准。
