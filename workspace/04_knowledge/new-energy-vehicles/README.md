# 热门新能源车型知识库

本目录用于帮助 Hermes Sales Agent 根据客户车型快速判断潜在销售机会，例如隐形车衣、太阳膜、改色膜、电动踏板、内饰保护、轮毂/卡钳等。

## 文件

- `50-model-sales-map.md`：给销售阅读和检索的车型销售机会地图。
- `new-energy-models.json`：结构化车型数据，便于 Agent 查询。
- `raw/热门新能源车型50个.md`：用户提供的原始资料。

## 使用边界

- 售价区间来自用户提供资料，只能作为车型层级参考，不作为当前报价或对客承诺。
- 车型颜色、配置、价格、热度会变化；正式对客前必须以厂家、门店或客户车辆实况为准。
- 销售点评用于内部启发，不能包装成市场排名、销量证明或真实客户案例。

## 推荐用法

- 客户报车型后，先判断车型定位和可能需求。
- 家庭 SUV/MPV：重点问老人小孩、后排隔热隐私、上下车便利。
- 高端/豪华车型：重点问原厂漆保护、车衣质保、内饰保护。
- 年轻化轿跑/小车：重点问改色、黑顶、轮毂/卡钳和个性化视觉。
- 所有报价、库存、施工排期必须回到门店确认。
