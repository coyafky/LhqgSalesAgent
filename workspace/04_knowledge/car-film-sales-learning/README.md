# 新手销售学习层 — 车膜（有膜有漾）

本目录是面向刚入职新手销售的“学习层”，由原始资料整理与精炼而成，方便新人快速上手并在与客户沟通时直接套用话术。

重要说明：
- 原始事实与细节以 [workspace/04_knowledge/car-wrim/ymyy-service-manual.jsonl](workspace/04_knowledge/car-wrim/ymyy-service-manual.jsonl) 为准。
- 本目录的内容是对原始知识的“销售学习化”呈现，用于：新人培训、Sales Agent 回答风格参考、FAQ 检索增强与 CRM 记录字段提示。
- 本目录内容不可用于：固定报价、活动承诺、库存承诺、施工排期承诺（这些需实时向门店/管理系统查询）。

使用说明：
- 每个 QA 包含：客户会怎么问、客户真实关心、销售先理解、可直接复制的对客话术、跟进补充点、不能说的话、CRM 记录建议、来源或来源状态。
- 涉及参数、质保、施工后注意事项的回答都已尽量标注来源页（若资料不足，会写明需要确认）。
