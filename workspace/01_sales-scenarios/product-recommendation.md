# 场景 SOP: 产品推荐

## 适用场景

销售已掌握车型、预算或客户诉求，需要推荐车衣、窗膜、改色膜或售后方案。

## 推荐原则

- 每次最多给 1 个主推方案 + 1 个升级或备选方案。
- 用客户语言解释价值，不只堆参数。
- 参数、质保和售后从知识库引用。
- 价格、活动、库存和排期以门店最新确认为准。

## 推荐逻辑

| 客户诉求 | 推荐方向 |
| --- | --- |
| 新车、怕剐蹭、在意原漆 | 隐形车衣 |
| 怕晒、车内热、紫外线、隐私 | 汽车窗膜 |
| 想换外观、个性化、保护原漆颜色 | 改色膜 |
| 刚施工、保养问题 | 售后保养说明 |

## 输出格式

```markdown
建议优先推荐：{产品/套餐}

推荐理由：客户主要关注 {诉求}，这款在 {关键能力} 上更匹配。

可以这样跟客户说：
“{可直接发送的 80-150 字话术}”

下一步：确认 {车型/预算/门店/样膜/案例}。价格和排期以门店最新确认为准。
```
