# 话术资产库

## 话术类型

- 首次接待话术。
- 需求诊断问题。
- 产品推荐话术。
- 透明价格解释。
- 异议处理话术。
- 议价缓和话术。
- 报价后跟进。
- 施工后提醒。
- 改色预览交付说明。

## 话术质量标准

- 客户能听懂，不堆术语。
- 不夸大、不绝对化、不攻击竞品。
- 有明确下一步。
- 价格、活动、库存、排期不编造。
- 能复制到微信或飞书直接发送。
